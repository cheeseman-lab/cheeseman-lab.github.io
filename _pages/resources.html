Resources

Brieflow & Large-Scale Screen Results

We developed Brieflow, an integrated computational pipeline for high-throughput analysis of optical pooled screening data. This tool enables efficient processing and analysis of large-scale CRISPR screens. We regularly publish results from our large-scale functional genomics screens, providing the scientific community with comprehensive datasets for exploring gene function and cellular processes.

Vesuvius

Vesuvius is a comprehensive interactive web portal designed for exploring image-based CRISPR screens of essential genes in human cells. This powerful tool allows researchers to browse and analyze phenotypic data from our microscopy-based genetic screens, providing insights into cellular organization and function.

Explore Vesuvius Database View Publication

Cell Cycle Knockouts Database

We have generated a collection of inducible CRISPR/Cas9-based knockout human cell lines targeting 209 genes with key roles in cell cycle functions. This resource enables precise studies of cell cycle structures, processes, and checkpoints. The database includes micrographs from analyses of knockout cell lines in HeLa cells.
